= Asciidoctor Numbered Captions Test

This file exercises `asciidoctor-numbered-captions` in AsciiDoc Zero-Network Preview.

Expected caption numbering:

* Chapter One starts at `1-1`.
* Nested sections inside Chapter One continue the Chapter One counters.
* Chapter Two resets each caption counter to `2-1`.
* Figure, table, and equation captions are counted independently.

== Sectioned Mode Quick Check

To verify section-based numbering instead, change the document header to:

----
:numbered-captions-numbering: sectioned
----

With sectioned mode, captions in nested sections use the current section number, such as `Figure 1.1-1`.

== Standard Mode Quick Check

To compare with Asciidoctor's standard caption numbering, change the document header to:

----
:numbered-captions-numbering: standard
----
