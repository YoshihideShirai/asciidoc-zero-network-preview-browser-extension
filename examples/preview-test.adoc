= AsciiDoc Preview Test
:toc:
:sectanchors:

This file exercises the local preview features in AsciiDoc Zero-Network Preview.

== Basic AsciiDoc

AsciiDoc inline formatting should render correctly:

* *bold*
* _italic_
* `monospace`
* link:README.md[local README link]

[NOTE]
====
Admonitions, lists, tables, diagrams, and math should all render without remote network access.
====

[%header,cols="1,2"]
|===
|Feature |Expected preview behavior

|AsciiDoc
|Rendered locally with Asciidoctor.js

|Images
|Remote image URLs are blocked, local image paths are rewritten for the VS Code Webview

|Diagrams
|Rendered from bundled JavaScript assets
|===
